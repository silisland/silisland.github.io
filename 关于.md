# 关于我

你好！欢迎来到我的技术博客。

## 博客简介

这是一个基于 [VitePress](https://vitepress.dev) 和 [Obsidian](https://obsidian.md) 构建的个人技术博客。

- 📝 所有内容使用 Markdown 编写
- 🔗 支持 Obsidian 双链语法
- 🚀 自动部署到 GitHub Pages
- 💬 集成 Giscus 评论系统

## 技术栈

| 技术 | 用途 |
| --- | --- |
| VitePress | 静态站点生成器 |
| Vue.js | 前端框架 |
| TypeScript | 类型安全的 JavaScript |
| Obsidian | 知识管理工具 |
| pnpm | 包管理器 |
| GitHub Pages | 静态托管 |

## 内容分类

本博客主要包含以下内容：

### 💻 技术笔记
记录开发过程中遇到的问题、解决方案和最佳实践。

### 🛠️ 工具分享
分享好用的开发工具、插件配置和使用技巧。

### 📚 知识管理
探索如何使用 Obsidian 和相关工具构建个人知识库。
